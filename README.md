# 🚀 Space Shooter — First Game

A classic 2D space shooter game built with Python and Pygame. Defend your spaceship against waves of enemies, survive as long as possible, and beat your high score!

## Features

- Player-controlled spaceship with left/right movement
- 6 enemies moving in formation, speeding up over time
- Bullet shooting with sound effects
- Score tracking and level progression
- Background music and explosion sounds
- Game Over screen when enemies reach your ship

---

## Made with 

- Python
- Pygame

Install Pygame with:
```bash
pip install pygame
```

##  How to Run

1. Clone the repository:
```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:
```bash
cd "Spaceship Game"
```

3. Run the game:
```bash
python "First game.py"
```

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ← Left Arrow | Move left |
| → Right Arrow | Move right |
| Space | Shoot |

---

##  How to Progress

- Every **10 points**, the game levels up
- Each level increases enemy speed
- Survive as long as you can!
